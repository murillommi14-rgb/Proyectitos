// AppIoT

Aplicación móvil Android para interconexión de dispositivos IoT.

## Descripción

Esta aplicación demuestra la interconexión entre actividades en una app móvil Android, simulando funcionalidades básicas de autenticación y gestión de cuentas. Incluye un splash screen, login, recuperación de clave y registro de cuenta, con navegación fluida y uso de AlertDialog para retroalimentación al usuario.

## Características

- **Splash Screen**: Pantalla de inicio con logo y mensaje de bienvenida.
- **Login**: Pantalla de inicio de sesión con validación básica.
- **Recuperar Clave**: Pantalla para recuperar contraseña vía email.
- **Registrar Cuenta**: Pantalla para crear una nueva cuenta con validación de campos.
- **Navegación**: Transición fluida entre actividades.
- **AlertDialog**: Mensajes de confirmación y error para acciones del usuario.

## Tecnologías Utilizadas

- **Lenguaje**: Kotlin
- **Framework**: Android SDK
- **Arquitectura**: Activities con navegación explícita
- **UI**: XML layouts con Material Design

## Instalación

1. Clona este repositorio.
2. Abre el proyecto en Android Studio.
3. Sincroniza Gradle y ejecuta en un emulador o dispositivo.

## Uso

- La app inicia con el Splash Screen.
- Navega a Login para iniciar sesión.
- Usa los botones para recuperar clave o registrar cuenta.
- Cada acción muestra un AlertDialog con el resultado simulado.

## Seguridad IoT

La app sigue estándares básicos de seguridad para aplicaciones móviles en entornos IoT, incluyendo validación de entradas y simulación de conexiones inalámbricas.
